## 📞 IF THINGS GO WRONG

**System doesn't work:**
- Show printed screenshots instead
- Continue presentation normally
- Offer live demo after

**You freeze or forget something:**
- Take a breath
- Check PRESENTATION_QUICK_REFERENCE.md
- Say: "Let me look at my notes for a second"
- Get back on track

**Boss interrupts with tough question:**
- Answer directly
- If you don't know: "That's a good question. Let me research and get back to you"
- Don't make up information

**Boss wants to delay decision:**
- Totally okay—ask what information would help
- Offer to send written analysis
- Ask when you can follow up
